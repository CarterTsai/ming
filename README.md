Image to PDF Generator
=============

### Usage

```
$> go run ming.go img2pdf create
```


### Help
```
$> go run ming.go img2pdf create -h
```

### Complie for Window

```
$> GOOS=windows GOARCH=386 go build -o ming.exe ming.go
```

```
$> GOOS=windows GOARCH=amd64 go build -o ming.exe ming.go
```
